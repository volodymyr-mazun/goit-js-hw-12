Завдання — Пошук зображень.<br>
Pixabay API підтримує пагінацію та надає параметри page і per_page. <br>
Додай в HTML-документі після галереї розмітку кнопки з текстом Load more.<br>
Якщо користувач дійшов до кінця колекції, ховай кнопку Load more.<br>
Зроби плавне прокручування сторінки після запиту і відтворення кожної наступної групи зображень. <br>
Використовуй метод window.scrollBy для прокрутки сторінки на дві висоти карточки галереї.<br>
